## Workflow

- **Add an idea:** `npm run new-idea "Your Video Title"`
- **Score an idea:** Open the file and ask Claude to score it against the methodology
- **Move to production:** `npm run new-project "Title"`

## Status key

| Status | Meaning |
|---|---|
| `idea` | Initial concept, not yet developed |
| `approved` | Scored and approved — ready for production planning |
| `filming` | On the filming schedule |
| `ready_to_edit` | Footage captured, ready to edit |
| `published` | Live |
| `archived` | Shelved |
